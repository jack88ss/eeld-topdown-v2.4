# 🚀 启动你的第一个教学设计项目

## 极简2步操作

### 第1步：进入项目目录
```powershell
cd 'C:\Users\Administrator\Desktop\myproject\eeld-top-down'
```

### 第2步：启动 Oh My OpenCode
```powershell
.\.opencode\start-ohmyopencode.ps1
```

## 🎯 就这么简单！

然后输入你的任务信息：
- **主题**：Unit 1 Goldilocks and the three bears
- **年级**：五年级
- **教材版本**：译林版

系统会自动执行9个阶段，在阶段4暂停让你确认核心大任务。

---

## 📋 完整示例

打开 PowerShell，复制粘贴：

```powershell
cd 'C:\Users\Administrator\Desktop\myproject\eeld-top-down'
.\.opencode\start-ohmyopencode.ps1
```

然后按提示输入：
```
主题: Unit 1 Goldilocks and the three bears
年级: 五年级
教材版本: 译林版
```

等待系统自动执行完成！
