# 🎯 快速开始指南

## 新手必读

本项目使用 **OpenCode + Oh My OpenCode** 构建，由 **Sisyphus** 作为主调度器管理教学设计工作流。

---

## ⚡ 3秒理解

```
你输入任务 → Sisyphus 自动调度 8 个 agents → 生成完整教案
```

---

## 🚀 立即开始（只需2步）

### 步骤1：启动

```powershell
cd 'C:\Users\Administrator\Desktop\myproject\eeld-top-down'
.\.opencode\start-ohmyopencode.bat
```

### 步骤2：输入任务

```
ulw Design lesson: Unit 1 Goldilocks, 五年级, 译林版
```

**完成！** Sisyphus 会自动执行 9 个阶段，15 分钟后生成完整教案。

## ❓ 常见问题

### Q: 什么是 Sisyphus？
A: Oh My OpenCode 的核心调度器，自动管理所有 sub-agents。

### Q: 什么是 ulw？
A: ultrawork 的缩写，激活 Sisyphus 的魔法关键词。

### Q: 需要手动运行每个阶段吗？
A: 不需要！输入 `ulw [任务]`，Sisyphus 自动完成一切。
