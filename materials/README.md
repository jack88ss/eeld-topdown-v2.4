# 教学材料文件夹

本文件夹用于存放教学设计所需的参考材料。

## 文件夹结构

```
materials/
├── textbooks/          # 教材文件
│   ├── student_book/   # 学生用书
│   └── teacher_book/   # 教师教学用书
├── standards/          # 课程标准
└── reference/          # 其他参考资料
```

## 支持的文件格式

- **PDF**: `.pdf` - 扫描版或电子版教材
- **Word文档**: `.doc`, `.docx` - 可编辑的文档
- **文本文件**: `.txt` - 纯文本格式
- **Markdown**: `.md` - Markdown格式文档

## 使用说明

### 1. 上传教材
将教材文件放入对应子文件夹：
- `textbooks/student_book/` - 学生教材
- `textbooks/teacher_book/` - 教师用书

### 2. 上传课程标准
将课程标准文档放入 `standards/` 文件夹：
- 《义务教育英语课程标准》
- 各年级英语教学大纲
- 学科核心素养文件

### 3. 上传参考资料
其他辅助材料放入 `reference/` 文件夹：
- 教学案例
- 教学论文
- 教学设计参考

## 文件命名建议

建议使用清晰的命名规范：
- `人教版PEP三年级上册_学生用书.pdf`
- `外研版四年级下册_教师用书.pdf`
- `义务教育英语课程标准2022版.pdf`
- `小学英语核心素养指南.md`

## 工作流程

1. **材料准备**: 用户上传相关文件到此文件夹
2. **材料审核**: 系统检查材料完整性，记录到 `state/MATERIAL_AUDIT.md`
3. **课程分析**: Curriculum Analyst 读取材料进行分析
4. **设计执行**: 基于真实材料进行教学设计

## 注意事项

- ⚠️ **版权声明**: 所有上传材料仅用于个人教学设计，请遵守版权法规
- 📝 **材料登记**: 所有材料来源将记录在 `state/SOURCES.md`
- 🔒 **隐私保护**: 如包含学生信息或敏感内容，请先脱敏处理
- 💾 **文件大小**: 建议单个文件不超过50MB，确保系统流畅运行
