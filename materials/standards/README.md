# 课程标准文件夹

存放国家及地方英语教学课程标准和指导文件。

## 核心文件

### 国家课程标准
- 《义务教育英语课程标准》（2022年版）
- 《义务教育英语课程标准》（2011年版）
- 《普通高中英语课程标准》

### 核心素养文件
- 英语学科核心素养框架
- 各学段核心素养要求
- 学业质量标准

## 课程标准主要内容

课程标准通常包含：
1. **课程性质与基本理念**
2. **课程目标** - 分级目标体系
3. **课程内容** - 语言知识、语言技能、文化意识、学习策略
4. **实施建议** - 教学建议、评价建议、教材编写建议
5. **附录** - 词汇表、语法项目表等

## 分级目标参考

小学阶段通常对应：
- **一级目标**: 小学3-4年级
- **二级目标**: 小学5-6年级

课程分析师会重点参考：
- 各年级的分级目标
