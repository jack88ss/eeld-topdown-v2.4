# 参考资料文件夹

存放教学设计的其他辅助参考材料。

## 可存放的资料类型

### 教学案例
- 优秀教学设计案例
- 名师课堂实录
- 教学比赛获奖作品

### 理论文献
- 英语教学法论文
- 小学生认知发展研究
- 课堂管理策略

### 教学资源
- 课堂游戏集锦
- 歌谣童谣资源
- 故事绘本资源

### 评价工具
- 评价量表
- 测试题库
- 评分标准

## 文件命名示例

```
小学英语TPR教学法应用研究.pdf
三年级英语课堂游戏100例.docx
优秀教案_Colors主题_王老师.md
小学生英语学习特点分析.txt
```

## 使用提示

这些参考资料将帮助：
- 课程分析师了解学生特点
- 内容设计师获取呈现灵感
- 活动设计师找到游戏创意
- 评估专家设计评价方案
