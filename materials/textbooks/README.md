# 教材文件夹

存放各版本小学英语教材。

## 子文件夹说明

### student_book/ - 学生用书
存放学生课本的电子版或扫描版。

### teacher_book/ - 教师教学用书
存放教师用书，通常包含：
- 教学目标和要求
- 教学建议和方法
- 课时安排
- 教学活动设计参考
- 练习答案

## 常见教材版本

- **人教版PEP** (人民教育出版社)
- **外研版** (外语教学与研究出版社)
- **译林版** (译林出版社)
- **北师大版** (北京师范大学出版社)
- **冀教版** (河北教育出版社)
- **湘少版** (湖南少年儿童出版社)

## 文件命名示例

```
人教版PEP_三年级上册_学生用书.pdf
外研版_四年级下册_教师用书.pdf
译林版_五年级上册_Unit1-Unit3.pdf
```
