# Assets 指南

- 保存文章中引用的图片或截图，命名为 `figure_01.png`, `figure_02.png` 等，与正文 `![[assets/figure_01.png]]` 对应。
- 每张图片需配套 `figures/figure_01.meta.json`，记录 `paragraph`、`description`、`source`、`license`、`captured_at`。
- 建议截图尺寸控制在 1200 像素以内，便于 Markdown 渲染。
- 若图片来自第三方，请在 meta 中注明出处与授权情况；如属自制截图，可填写 "author-self"。
- 发布前由 publisher 逐一核对：图片存在、meta 信息完整、正文描述同步。
