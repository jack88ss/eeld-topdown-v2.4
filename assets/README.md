# 教学资源文件夹

本文件夹用于存放教学设计生成的各类资源文件。

## 资源类型

### 1. 数字资源
- **PPT大纲**：`ppt_outline.md` - 课件结构和内容要点
- **图片素材**：`image_01.png`, `image_02.png` 等
- **音频文件**：`audio_01.mp3` - 单词发音、对话录音
- **视频资源**：`video_01.mp4` - 动画、视频片段

### 2. 打印资源
- **单词卡片**：`flashcards.pdf` - 可打印的单词卡片
- **图片卡片**：`picture_cards.pdf` - 教学图片卡片
- **学习单**：`worksheet.pdf` - 课堂练习单
- **评价表**：`assessment_form.pdf` - 评价量表

### 3. 板书设计
- **板书示意图**：`board_design.png` - 板书布局示意
- **关键词卡片**：供板书使用的关键词

## 文件命名规范

### 图片文件
```
<类型>_<序号>_<描述>.png

示例：
- word_card_01_apple.png
- picture_02_school.png
- board_design_colors.png
```

### 音频文件
```
<类型>_<序号>_<内容>.mp3

示例：
- word_pronunciation_apple.mp3
- dialogue_01_greetings.mp3
- song_color_song.mp3
```

### 文档文件
```
<类型>_<主题>.pdf

示例：
- flashcards_animals.pdf
- worksheet_colors.pdf
- assessment_form_unit3.pdf
```

## 资源来源记录

所有资源的来源必须在 `state/SOURCES.md` 中记录：

- **自制资源**：标注 "自制" 或 "原创"
- **网络资源**：记录完整URL和访问时间
- **教材配套**：注明教材名称和出版信息
- **第三方素材**：记录来源和版权信息

## 版权声明

### 允许使用的资源
- ✅ 自己创作的原创资源
- ✅ 明确标注为免费教育资源的素材
- ✅ 教材配套的官方资源
- ✅ 获得授权的第三方资源

### 禁止使用的资源
- ❌ 未经授权的商业图片
- ❌ 侵犯版权的音视频
- ❌ 来源不明的素材

## 资源准备建议

### 图片资源
- **格式**：PNG（透明背景）或 JPG
- **尺寸**：1200px 以内，适合课堂投影
- **清晰度**：高清，避免模糊
- **风格**：符合学生年龄特点，色彩鲜艳

### 音频资源
- **格式**：MP3
- **音质**：清晰，无杂音
- **语速**：适合学生水平
- **时长**：单个音频不超过3分钟

### PPT资源
- **格式**：PowerPoint (.pptx) 或 PDF
- **风格**：简洁明了，避免过度装饰
- **字体**：大字号，易于阅读
- **动画**：适度使用，不影响教学

## 资源协调流程

1. **Resource Coordinator** 设计资源清单
2. 在 `state/RESOURCE_LIST.md` 中详细列出
3. 将资源文件放入本文件夹
4. 在 `state/SOURCES.md` 记录来源
5. **Package Publisher** 打包时检查完整性

## 注意事项

- 📁 保持文件夹整洁，使用清晰的命名
- 📝 必须记录所有资源来源
- ⚖️ 严格遵守版权法规
- 🔍 Package Publisher 会检查资源完整性
- 💾 建议定期备份资源文件
