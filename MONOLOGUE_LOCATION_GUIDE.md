# 代理独白记录位置指南

## 核心原则

**统一记录位置**：所有代理的内心独角戏、工作过程、交接记录都必须写入 `state/LOG.md` 文件。

## 为什么不在代理配置文件中记录？

1. **单一事实源**：`state/LOG.md` 是所有执行记录的唯一位置
2. **时间线清晰**：所有代理活动按时间顺序排列
3. **便于追踪**：一个文件即可看到完整的创作过程
4. **避免混淆**：配置文件（.claude/agents/*.md）只定义行为规范，不存储执行记录

## 记录结构

### 1. 代理配置文件（.claude/agents/*.md）
**作用**：定义代理的行为规范、角色定位、任务要求
**内容**：
- 角色定位（性格、口头禅、关注点）
- 任务要求（输入、输出、验证）
- 记录格式模板（但不是实际记录）

### 2. 执行日志文件（state/LOG.md）
**作用**：记录所有代理的实际执行过程
**内容**：
- 每个代理的五幕独角戏
- 交接验证记录
- 时间戳和签名

## 标准记录格式

在 `state/LOG.md` 中：

```markdown
===== 2025-01-18 10:00:00 UTC | Stylist 登场 =====

【Stylist 登场】
我是风格分析师，负责把握文章的语气和节奏。今天的任务是分析用户提供的主题...

【观察输入】
- 主题：AI开发工具
- 样例：13篇博客
- 要求：3000字深度文章

【内心独白】
这个技术主题需要平衡专业性和亲切感...
看了样例，作者喜欢用生活化比喻...

【决策过程】
决定沿用现有风格缓存，因为：
1. 样例未更新
2. 风格适合当前主题

【执行动作】
✅ 复用 state/STYLE_PROFILE.md
✅ 更新 state/WRITING_SPEC.md
⚠️ 标记风险：技术术语需简化

产出物：
- state/STYLE_PROFILE.md (未修改)
- state/WRITING_SPEC.md (v2.0)

【交接验证】
□ 风格缓存有效：YES ✓
□ 所有字段完整：YES ✓
□ 无TODO项：YES ✓

【Stylist 退场】
风格基调已定，交给Coordinator处理素材。
记住保持亲切的对话感！

签名：stylist_20250118_1000UTC
=====

===== 2025-01-18 10:15:00 UTC | Coordinator 登场 =====

【Coordinator 登场】
我是项目协调员，负责整理资源和制定计划。
收到Stylist的风格定位，现在来看看素材情况...

[继续记录...]
=====
```

## 代理间对话记录

当代理需要沟通时，也在 `state/LOG.md` 中记录：

```markdown
【Coordinator → Researcher】
"主题涉及GitHub新工具，需要你深挖：
- 工具采用率
- 竞品对比
- 社区反馈"

【Researcher 回应】
"收到！重点关注批评声音，保持平衡报道。"
```

## 常见错误

❌ **错误**：在 `.claude/agents/writer.md` 中写入实际执行记录
```markdown
# .claude/agents/writer.md
【Writer 登场】
今天我写了一篇关于... # ❌ 不要在这里记录
```

✅ **正确**：在 `state/LOG.md` 中记录实际执行
```markdown
# state/LOG.md
===== 2025-01-18 15:00:00 UTC | Writer 登场 =====
【Writer 登场】
今天我写了一篇关于... # ✓ 在这里记录
```

## 查找记录的方法

```bash
# 查看某个代理的所有记录
grep -A 50 "Writer 登场" state/LOG.md

# 查看特定时间的记录
grep "2025-01-18" state/LOG.md

# 查看所有交接验证
grep "交接验证" state/LOG.md

# 查看返工记录
grep "REWORK" state/LOG.md
```

## 记录完整性检查

每个代理执行后必须在 `state/LOG.md` 中有：
1. ✅ 登场时间戳
2. ✅ 五幕独角戏
3. ✅ 产出物列表
4. ✅ 交接验证结果
5. ✅ 责任签名

## 总结

- **配置文件**（.claude/agents/*.md）：定义"应该怎么做"
- **日志文件**（state/LOG.md）：记录"实际做了什么"

所有实际执行记录都在 `state/LOG.md`，这是唯一的事实源！
