# 活动设计方案

**设计者**：Activity Designer  
**日期**：2026-01-21  
**基于**：Big Task Proposal - Three Bears 新家设计师  
**课时**：40分钟  

---

## 🎯 整体设计理念

**核心任务驱动**：围绕"为 Three Bears 设计新家"这个真实情境，设计 Pre-While-Post 三阶段活动链，让学生在完成任务的过程中自然习得和运用 There is/are 句型。

**语言习得路径**：
```
输入理解 (Pre) → 控制性练习 (While-1) → 半控制练习 (While-2) → 自由输出 (Post)
```

---

## 📋 完整活动链（40分钟）

### **阶段 1: Pre-task 预备活动（10分钟）**

#### Activity 1.1: Warm-up - Feelings Guessing Game (3分钟)

**目标**：激活情感词汇，回顾故事情节

**步骤**：
1. 教师出示 Goldilocks 图片，做出表情动作
2. 学生猜测：Is she hungry/thirsty/tired/afraid?
3. 快速复述故事：Goldilocks is in the forest. She is hungry...

**互动形式**：全班问答

**语言焦点**：hungry, thirsty, tired, afraid

---

#### Activity 1.2: Story Review - Picture Sequencing (4分钟)

**目标**：梳理故事脉络，复习 There is/are

**步骤**：
1. 展示4张打乱顺序的故事图片：
   - Forest → House → Soup → Beds → Bears
2. 学生排序并说出关键句：
   - There is a house in the forest.
   - There is some soup on the table.
   - There are three beds in the room.
   - There are three bears!

**互动形式**：个人思考 + 全班核对

**语言焦点**：There is/are 的初步回顾

---

#### Activity 1.3: Vocabulary Activation - Home Items Brainstorm (3分钟)

**目标**：激活家具词汇，为大任务做准备

**步骤**：
1. PPT 展示不同房间图片（living room, bedroom, kitchen）
2. 学生brainstorm家具名称：bed, table, chair, sofa, TV...
3. 教师板书词汇库，分类呈现

**互动形式**：头脑风暴 + 板书整理

**产出**：黑板上的 Word Bank

---

### **阶段 2: While-task 任务实施（25分钟）**

#### Activity 2.1: Language Focus - There is/are Mini-lesson (5分钟)

**目标**：明确语法规则，提供语言支架

**步骤**：
1. 展示示例句：
   - There is **a bed** in the bedroom. (单数)
   - There are **three beds** in the room. (复数)
2. 学生总结规则：1个用is，2个或更多用are
3. 快速练习：
   - There ___ a table. (is)
   - There ___ two chairs. (are)
   - There ___ some books. (are)

**互动形式**：教师讲解 + 快速练习

**语言焦点**：There is vs There are 的区别

---

#### Activity 2.2: Controlled Practice - Describe Teacher's Room (3分钟)

**目标**：控制性练习，巩固句型

**步骤**：
1. 展示教师卧室图片
2. 学生用 There is/are 描述：
   - There is a bed.
   - There are two lamps.
   - There is a computer on the desk.
3. 教师纠正错误，强调单复数

**互动形式**：个人 + 举手发言

**产出**：6-8个正确句子

---

#### Activity 2.3: Semi-controlled Practice - Pair Description (4分钟)

**目标**：半控制练习，同伴互动

**步骤**：
1. 学生 A 看房间图片（教师提供），描述给学生 B
2. 学生 B 根据描述画出房间布局
3. 对比原图，检查描述是否准确
4. 交换角色

**互动形式**：两人一组

**语言支架**：句型卡片

---

#### Activity 2.4: Big Task - Design Bears' New Home (13分钟)

**目标**：核心任务，综合应用语言

**步骤**：

**Step 1: 小组讨论 (4分钟)**
- 4人一组，决定设计几个房间
- 分配任务：谁负责哪个房间
- 头脑风暴：每个房间需要什么家具

**Step 2: 绘制平面图 (6分钟)**
- A3纸分区域画房间
- 画出家具并标注英文名称
- 添加装饰（窗户、植物、图画）

**Step 3: 准备介绍 (3分钟)**
- 每人写下要说的2-3句话
- 组内练习，互相纠正发音
- 确定展示顺序

**互动形式**：小组合作

**教师巡视**：
- 提供词汇支持
- 纠正 There is/are 错误
- 鼓励添加细节描述

---

### **阶段 3: Post-task 展示评价（5分钟）**

#### Activity 3.1: Group Presentation (3分钟)

**目标**：展示成果，真实交际

**步骤**：
1. 每组派代表展示作品（贴在黑板上）
2. 轮流介绍房间：
   - Group A: "In Papa Bear's bedroom, there is a big bed..."
   - Group B: "In the living room, there are two sofas..."
3. 其他学生认真倾听，准备提问或点评

**互动形式**：小组展示 + 全班倾听

**语言输出**：每组 6-8 句 There is/are

---

#### Activity 3.2: Gallery Walk & Voting (2分钟)

**目标**：同伴评价，提升参与感

**步骤**：
1. 学生走动欣赏各组作品
2. 每人在最喜欢的设计旁边贴星星贴纸
3. 统计票数，颁发"最贴心设计奖"

**互动形式**：自由走动 + 投票

**情感目标**：成就感和相互欣赏

---

#### Activity 3.3: Summary & Homework Assignment (1分钟)

**总结**：
- What did we learn today? (There is/are, home words)
- What did we do? (Designed a home for Bears)

**作业**：
- 用 There is/are 写 5 句话介绍自己的房间
- 画出自己理想房间的平面图

## 🎯 互动形式分布

```
全班互动：30%  (Warm-up, Review, Mini-lesson, Summary)
小组合作：40%  (Big Task - 13分钟)
两人一组：15%  (Pair Description)
个人思考：15%  (思考时间、作业)
```

**设计理念**：
- 避免教师独白，增加学生参与
- 小组合作降低焦虑，提升信心
- 多样化互动保持注意力

## 🎨 差异化支持策略

### For 学习困难学生

**降低难度**：
- 提供已画好的房间轮廓纸，只需添加家具
- 提供填空式句型卡：There ___ a ___ in the ___.
- 允许使用图片+指认方式（而非完整句子）
- 小组角色：负责绘画，其他成员负责语言

**额外支持**：
- 教师巡视时优先关注
- 提供一对一词汇提示
- 允许使用电子词典

---

### For 优秀学生

**增加挑战**：
- 描述家具的位置关系：next to, between, in front of, behind
- 增加描述细节：颜色(red sofa)、大小(big/small)、材质(wooden table)
- 设计多个房间并串联介绍
- 尝试使用复合句：There is a bed and there are two lamps.

**拓展任务**：
- 为 Goldilocks 也设计一个家，对比两个家的不同
- 写一段话介绍设计理念（Why did you choose these things?)

---

### For 中等水平学生

**标准任务**：
- 按照常规流程完成设计
- 达到 6-8 句 There is/are 描述
- 参与小组讨论和展示

---

## ⚠️ 可能的问题与应对

| 问题 | 应对策略 |
|------|---------|
| **学生不知道画什么** | 展示示例图片，头脑风暴清单，允许模仿 |
| **There is/are 混淆** | 板书口诀"1个is，多个are"，巡视纠正 |
| **不敢开口展示** | 允许小组一起说，或只说1-2句 |
| **时间不够** | 简化绘画（草图即可），重点放语言 |
| **小组分工不均** | 明确角色：设计师、绘图师、文字师、发言人 |
| **词汇不足** | 提供图片词汇卡，允许查看教材 |
| **纪律问题** | 设置小组积分制，安静合作加分 |

## 🎯 学习成果评估

### 形成性评估（课堂中）

**观察要点**：
- ✅ 学生能否区分 There is 和 There are
- ✅ 学生能否用英语描述 4-5个房间物品
- ✅ 学生能否在小组中积极参与
- ✅ 学生能否完整表达 2-3 个句子

**记录方式**：
- 教师巡视时记录重点问题
- 拍照记录小组作品
- 记录典型错误用于讲评

---

### 终结性评估（课后）

**作业评估标准**：
- ✅ 能写出 5 句 There is/are 句子
- ✅ 单复数使用正确
- ✅ 词汇拼写基本正确
- ✅ (可选) 配上房间简图

---

## 💡 教学反思建议

**课后思考**：
1. 学生是否真正理解 There is/are 的区别？
2. 小组活动时间分配是否合理？
3. 哪些学生需要额外辅导？
4. 学生的创意和语言输出是否达到预期？

**改进方向**：
- 如果语法错误多 → 下节课增加针对性练习
- 如果时间不够 → 简化Pre-task，延长While-task
- 如果学生兴趣高 → 下次可以做"Dream House"主题拓展

---

## ✅ 活动设计总结

### 亮点

1. ✅ **任务真实有趣**：为 Bears 设计新家，情境自然
2. ✅ **语言聚焦明确**：始终围绕 There is/are
3. ✅ **互动形式多样**：全班、小组、两人、个人都有
4. ✅ **差异化支持充分**：照顾不同水平学生
5. ✅ **成果可视化**：手绘作品可展示和留存

### 对齐核心理念

- ✅ **任务驱动**：真实情境，完成任务 = 学会语言
- ✅ **自上而下**：从整体任务到具体语言点
- ✅ **学生中心**：学生是设计师，教师是支持者
- ✅ **输出导向**：最终产出是语言 + 作品
