## 📝 大任务详细信息

### 任务情境
**真实场景**：班级故事分享会，学生以小组形式讲述故事给同伴听。

**交际目的**：清晰描述故事情节与人物感受，让听众理解故事发展。

### 任务目标
**学生要完成什么**：制作 4-5 幅故事板并用英语连续讲述。

**成功标准**：
- [x] 讲述 6-8 句，顺序正确
- [x] 至少 2 处 There is/are
- [x] 至少 2 个感受词表达

### 任务成果
**最终产出**：故事板（图+句）与现场讲述展示。

**成果用途**：用于班级故事展示墙或课堂分享。

---

## 🛠️ 完成大任务需要的语言工具（初步）

### 核心句型
1. There is/are ___ in the ___.
2. This ___ is too ___ / just right.
3. She is ___ now.

### 关键词汇
- 场景与物品：forest, house, room, soup, bed, bear
- 感受与描述：hungry, thirsty, tired, afraid, hot, cold, hard, soft

### 交际策略
- 用 First/Then/Next/Finally 保证顺序
- 不会的词用同伴提示卡或图片提示
