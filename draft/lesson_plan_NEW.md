## 📚 教材分析

### 单元主题
Unit 1 Goldilocks and the Three Bears 讲述经典童话故事，通过故事情境教授 **There is/are** 句型和情感形容词。

### 课时定位
本课时为第1课时（Story time），重点是故事理解和 There is/are 句型的初步学习与运用。

## 🎯 教学目标

### 1. 语言知识目标
- **词汇**：能听懂、会说、会读 forest, house, room, soup, bear, hungry, thirsty, tired, afraid, hard, soft, hot, cold
- **句型**：能理解并运用 "There is..." 和 "There are..." 描述场景和物品

### 2. 语言技能目标
- **听**：能听懂故事大意和关键信息
- **说**：能用 There is/are 描述房间和物品，能简单复述故事片段
- **读**：能朗读故事文本

### 3. 学习策略目标
- 能通过小组合作完成设计任务
- 能在真实情境中运用所学语言

### 4. 情感态度目标
- 培养对英语故事的兴趣
- 理解"尊重他人财产"的道德寓意

## 🔍 学情分析

### 学生特征
- **年龄**：10-11岁，五年级学生
- **认知水平**：具备一定逻辑思维，喜欢故事情节
- **注意力**：能保持20-25分钟高效注意力

### 已有基础
- 已学过基本家具词汇（bed, table, chair）
- 三四年级接触过简单的 There be 句型
- 熟悉 Goldilocks 中文故事背景

### 学习难点
- There is 与 There are 的区别（特别是不可数名词）
- soft/hard/thirsty 的准确发音
- 在真实情境中灵活运用句型

## 📖 教学重难点

### 教学重点
1. 理解故事内容，复述故事片段
2. 正确使用 There is/are 句型描述物品
3. 掌握核心词汇：hungry, thirsty, tired, afraid, hard, soft

### 教学难点
1. 区分 There is (单数/不可数) 和 There are (复数)
2. 在真实任务中综合运用语言知识
3. 自然流畅地进行口语表达

## ⏰ 教学流程（40分钟）

### **Step 1: Pre-task 预备活动（10分钟）**

#### Activity 1: Warm-up - Feelings Guessing (3分钟)

**教学活动**：
1. 教师展示 Goldilocks 不同表情的图片
2. 学生猜测：Is she hungry? / Is she thirsty? / Is she tired? / Is she afraid?
3. 快速复习故事：Goldilocks is in the forest. She is hungry and thirsty.

**设计意图**：激活情感词汇，唤起故事背景知识

---

#### Activity 2: Story Review - Picture Sequencing (4分钟)

**教学活动**：
1. PPT展示4张打乱顺序的故事场景图
   - Forest → House → Soup → Beds → Bears
2. 学生排序并说出关键句：
   - There is a house in the forest.
   - There is some soup on the table.
   - There are three beds in the room.
3. 全班核对答案

**设计意图**：梳理故事脉络，初步复习 There is/are

---

#### Activity 3: Vocabulary Brainstorm (3分钟)

**教学活动**：
1. PPT展示不同房间图片（living room, bedroom, kitchen）
2. 学生brainstorm家具名称：bed, table, chair, sofa, TV, bookshelf...
3. 教师板书词汇，分类整理

**板书**：
```
Rooms: bedroom, living room, kitchen, bathroom
Furniture: bed, table, chair, sofa, TV, desk, bookshelf, lamp
```

**设计意图**：激活已学词汇，为大任务做准备

---

### **Step 2: While-task 任务实施（25分钟）**

#### Activity 4: Grammar Focus - There is/are (5分钟)

**教学活动**：
1. 展示示例句并突出重点：
   - There is **a bed** in the bedroom. (单数)
   - There is **some soup** on the table. (不可数)
   - There are **three beds** in the room. (复数)
   
2. 引导学生总结规则：
   - Teacher: When do we use "is"? When do we use "are"?
   - Students: 1个用is，2个或更多用are！
   
3. 快速练习（填空）：
   - There ___ a table. (is)
   - There ___ two chairs. (are)
   - There ___ some books on the shelf. (are)

**板书**：
```
There is + a/an/some + 单数/不可数名词
  ↳ There is a bed.
  ↳ There is some soup.

There are + 复数名词
  ↳ There are three beds.
```

**设计意图**：明确语法规则，提供语言支架

---

#### Activity 5: Controlled Practice - Describe Teacher's Room (3分钟)

**教学活动**：
1. PPT展示教师卧室图片
2. 学生用 There is/are 描述：
   - S1: There is a bed.
   - S2: There are two lamps on the table.
   - S3: There is a computer on the desk.
3. 教师纠正典型错误，强调单复数

**设计意图**：控制性练习，巩固句型

---

#### Activity 6: Pair Work - Description Game (4分钟)

**教学活动**：
1. 学生A看房间图片（教师提供），描述给学生B
2. 学生B根据描述画出房间布局
3. 对比原图，检查准确性
4. 交换角色重复活动

**语言支架**：提供句型卡片辅助

**设计意图**：半控制练习，增强互动性

---

#### Activity 7: 🎨 **Big Task - Design Bears' New Home** (13分钟)

**任务情境**：
Goldilocks 闯入了 Bears 的旧家，Bears 决定搬新家。同学们作为"家居设计师"，帮助 Three Bears 设计一个温馨舒适的新家！

**Step 1: 小组讨论 (4分钟)**
- 4人一组
- 讨论：Papa Bear / Mama Bear / Baby Bear 需要哪些房间？每个房间需要什么家具？
- 分配任务：谁负责哪个房间

**Step 2: 绘制平面图 (6分钟)**
- A3纸上画出房间布局
- 画出家具并标注英文名称
- 添加装饰细节（窗户、植物、图画）

**Step 3: 准备介绍 (3分钟)**
- 每人写下要说的2-3句话
- 组内练习发音
- 确定展示顺序

**教师巡视**：
- 提供词汇支持（参考Word Bank）
- 纠正 There is/are 错误
- 鼓励添加细节描述

**设计意图**：核心任务，综合运用语言，培养创造力和协作能力

---

### **Step 3: Post-task 展示评价（5分钟）**

#### Activity 8: Group Presentation (3分钟)

**教学活动**：
1. 各组展示作品（贴在黑板上）
2. 每组派代表介绍（每组约30秒）：
   - Group A: "In Papa Bear's bedroom, there is a big bed and there are two lamps."
   - Group B: "In the living room, there is a sofa and there are three chairs."
3. 其他学生倾听并准备投票

**设计意图**：真实语言输出，展示学习成果

---

#### Activity 9: Gallery Walk & Voting (2分钟)

**教学活动**：
1. 学生走动欣赏各组作品
2. 每人在最喜欢的设计旁边贴星星贴纸
3. 统计票数，颁发"最贴心设计奖"🏆

**设计意图**：同伴评价，提升参与感和成就感

---

### **Step 4: Summary & Homework (1分钟)**

#### Summary
**Teacher**: What did we learn today?
**Students**: 
- We learned "There is" and "There are".
- We learned home words: bedroom, living room, bed, sofa...

**Teacher**: What did we do?
**Students**: We designed a new home for Three Bears!

---

#### Homework
1. 用 There is/are 写5句话介绍自己的房间
2. 画出自己理想房间的简单平面图

**示例**：
```
In my bedroom, there is a bed.
There are two windows.
There is a desk and there are many books on it.
...
```

## 💡 教学反思

### 课后思考问题
1. 学生是否真正理解 There is 和 There are 的区别？
2. 小组活动时间分配是否合理？
3. 哪些学生需要额外辅导？
4. 学生的语言输出是否达到预期？

### 改进方向
- 如果语法错误多 → 增加针对性练习
- 如果时间不够 → 简化Pre-task
- 如果学生兴趣高 → 拓展"Dream House"主题

## 📖 附录

### 附录1：差异化支持策略

**For 学习困难学生**：
- 提供已画好的房间轮廓纸
- 提供填空式句型卡
- 允许使用图片+指认方式
- 小组分工：负责绘画

**For 优秀学生**：
- 描述位置关系（next to, between, in front of）
- 增加描述细节（颜色、大小）
- 设计多个房间
- 写一段设计理念

---

### 附录2：常见问题与应对

| 问题 | 应对策略 |
|------|---------|
| 学生不知道画什么 | 展示示例图片，头脑风暴清单 |
| There is/are 混淆 | 板书口诀，巡视纠正 |
| 不敢开口展示 | 允许小组一起说 |
| 时间不够 | 简化绘画，重点放语言 |
